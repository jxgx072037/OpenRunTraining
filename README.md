# Strava运动数据可视化

这是一个基于Flask的Web应用，用于展示和分析Strava的运动数据。通过Strava API获取运动数据，并使用现代化的图表和地图进行可视化展示。

## 功能特点

- 🔐 Strava OAuth2.0认证登录
  - 友好的欢迎页面介绍
  - 用户主动授权流程
  - 自动令牌刷新机制
- 📊 运动数据可视化
  - 配速曲线
  - 心率数据
  - 步频数据
  - 海拔变化
- 🗺️ 运动轨迹地图展示
  - 年度活动轨迹汇总
  - 智能聚焦最常运动区域
  - 简洁清晰的地图风格
  - 交互式轨迹显示
- 📈 交互式数据探索
- 📱 响应式设计
- 🌤️ 历史天气数据分析
  - 过去10年同期天气数据
  - 温度趋势图表
  - 湿度、降水、风速等详细信息

## 技术栈

- 后端：Python Flask
- 前端：
  - Chart.js - 数据可视化
  - Leaflet.js - 地图展示
  - Bootstrap 5 - UI框架
- 数据：
  - Strava API v3
  - Visual Crossing Weather API
- 部署：
  - Nginx - 反向代理服务器
  - Gunicorn - WSGI服务器
  - Let's Encrypt - SSL证书

## 配置说明

1. 创建`.env`文件并设置以下环境变量：
```
STRAVA_CLIENT_ID=你的客户端ID
STRAVA_CLIENT_SECRET=你的客户端密钥
STRAVA_REDIRECT_URI=你的回调地址
SECRET_KEY=你的密钥
```

2. 安装依赖：
```bash
pip install -r requirements.txt
```

3. 运行应用：
```bash
python app.py
```

4. 生产环境部署：
```bash
# 使用gunicorn启动应用
gunicorn -c gunicorn_config.py app:app

# 或使用systemd服务
sudo systemctl start openrun
```
